Predictive Brain Lab

A real-time predictive-coding toy model with precision-weighted prediction error, homeostatic regulation, and accumulating allostatic load.

Live signal simulation

How to use this model

  1. Pick a mode or preset scenario.
  2. Adjust precision, noise, and load.
  3. Track confidence, error, and balance over time.

Blue pulses represent top-down predictions; amber pulses represent bottom-up (precision-weighted) prediction error.

Under the hood: a single latent interoceptive state (x) generates a noisy sensory sample (y). Beliefs (μ) update via precision-weighted prediction errors; regulation acts on x; load accumulates from sustained error and control effort.

Start with a scenario preset, then tune sliders to test alternative trajectories.
